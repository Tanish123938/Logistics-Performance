# Logistics Performance Dashboard (Power BI)
## Project Overview

This project focuses on analyzing logistics and supply chain data to monitor delivery performance, shipment status, costs, and regional efficiency.
The dashboard helps stakeholders make data-driven decisions by providing clear insights into operational performance.

## Objectives

- Track on-time vs delayed deliveries
- Analyze shipment volume and logistics cost
- Identify high-performing and low-performing regions
- Monitor overall supply chain efficiency

## Tools & Technologies Used

- Power BI – Data modeling & dashboard creation
- Microsoft Excel / CSV – Data source
- Power Query – Data cleaning & transformation
- DAX – Calculated measures and KPIs

## Key Dashboard Features

- Total Shipments KPI
- On-Time Delivery %
- Delayed Shipments Analysis
- Cost Analysis by Region
- Shipment Status Breakdown
- Region-wise Performance Comparison
- Interactive filters & slicers for better analysis

## Key Insights

- Certain regions show consistently higher delivery delays
- Logistics costs vary significantly by shipment type

## Conclusion

This dashboard provides a clear and actionable view of logistics operations, enabling better planning, cost control, and service improvement.

On-time delivery rate improves with optimized routing

High shipment volume does not always mean higher efficiency
